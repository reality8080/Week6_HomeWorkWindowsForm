### 5. **Hướng Dẫn Sử Dụng**
Hướng dẫn cách sử dụng dự án, có thể kèm theo hình ảnh minh họa.
```markdown
## Hướng Dẫn Sử Dụng
- **Tính năng chính:**
  - Thêm/sửa/xóa sản phẩm.
  - Quản lý hóa đơn.
- **Cách dùng:**
  - Nhấn vào nút "Thêm" để thêm sản phẩm mới.
  - Điền thông tin và nhấn "Lưu".
